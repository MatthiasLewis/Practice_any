"雖然 OpenAI 不打算開放 ChatGPT 原始碼，甚至被批評已淪為微軟賺錢工具，但卻無法阻止其他生成式 AI 聊天機器人的開源之路。去年募資 1 億美元的 AI 新創公司 Hugging Face 正式推出自家版聊天機器人 HuggingChat，以作為當紅 ChatGPT 的開放原始碼替代品。"
"HuggingChat 採用由 Open Assistant 開發的 AI 模型，Open Assistant 是德國非營利性組織 LAION 旗下專案，專門負責創建資料集，以便用來訓練 Stable Diffusion 文字轉圖像 AI 模型。原則上，這款開源聊天機器人和 ChatGPT 一樣能用於編寫程式碼、起草電子郵件或撰寫饒舌歌曲歌詞。使用者可以透過 Web 介面進行試用，也可透過 Hugging Face 提供的 API 與現有應用程式及服務相整合。"
"該聊天機器人也和時下當紅 ChatGPT 一樣會有回答不夠精準，甚至編造事實的狀況出現。例如連問兩次 2020 年誰贏得美國總統大選，結果兩次回答竟然不一樣。但幸好 HuggingChat 也支援能固守安全及道德底線的過濾器，只要問它涉及非法或種族歧視等有爭議的問題時，它會拒答。"
"Open Assistant 在自家 GitHub 專頁上展望指出，他們想打造未來 AI 助理，不僅能寫求職信，還能從事諸如善用 API、動態研究資訊等有意義的工作，甚至任何人都能對該模型進行擴充與個性化。想要實現這點，該模型就必須滿足小而美（高效）的要求，以便能在消費性硬體上順暢運行。"
"（首圖來源：科技新報）"
